Mi Primera Calculadora en Python

Este es mi primer proyecto real programando en Python.

Qué hace este programa?
Una suma simple
Interactiva: eliges que hacer (sumar, restar, multiplicar o dividir).
Gracias a un bucle, puedes hacer todos los cálculos que quieras sin tener que reiniciar el programa a cada rato.
Si intentas dividir por cero (que es un error típico), el programa te avisa en lugar de fallar.
